테스트 베드 신청

'테스트베드 신청하기' 버튼을 누르면 신청 화면으로 이동됩니다.

테스트베드 신청
01
테스트베드 신청

테스트베드 신청 버튼을 클릭하여 신청서를 작성합니다.

신청 현황 확인

마이페이지를 통해 신청 현황을 확인합니다.

디바이스 배치

테스트할 디바이스 위치를 지정하여 배치합니다.

디바이스 연동
02
프로토콜 규격 확인

스마트시티 테스트베드에서 지원 가능한 프로토콜 규격을 확인합니다.

디바이스 연결

프로토콜 규격에 따라 디바이스 연결합니다.

디바이스 관리
03
디바이스 등록

디바이스 정보를 입력하여 등록합니다.

측정 항목 설정

디바이스의 측정 항목을 설정합니다.

이벤트 설정

특정 조건에 부합하는 상황 발생 시 확인 가능하도록 이벤트를 설정합니다.

미세먼지 모니터링
04
실시간 모니터링

테스트 디바이스의 실시간 측정데이터를 상용 디바이스의 측정데이터과 비교하여 모니터링합니다.

측정값 보기

기간별 분/시/일/1주일/30일 단위의 측정 값 추이를 확인하고, 필요한 데이터를 다운로드합니다.

디바이스 상태

디바이스의 데이터 수집 건수, 성공율, 이벤트 발생 이력 등의 상태 정보를 확인합니다.
